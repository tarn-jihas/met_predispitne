Grupa zadataka 2

Zadatak iz GRUPE 2 birate prema sledećoj formuli: brIndeksa % 15 + 1. Na primer: 5231 % 15 = 11, 11+1=12, student radi zadatak 12.

GRUPA 2:
1. Napisati JS program koji sortira niz elemenata koristeći Bubble sort. Kreirajte sami proizvoljan niz od minimum 7 elemenata.
2. Napisati JS program koji ispisuje indekse brojeva koji su veći od prosečne vrednosti brojeva u nizu. Kreirajte sami proizvoljan niz od minimum 7 elemenata.
3. Napisati JS program koji pronalazi prvih N elemenata niza. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Pronađite prvih 3 elementa niza.
4. Napisati JS program koji pronalazi najčešću vrednost u nizu. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Prikažite koji je element u pitanju i koliko puta se ponavlja.
5. Napisati JS program koji pronalazi i uklanja duplikate iz niza (ignorisati case sensitivity). Kreirajte sami proizvoljan niz od minimum 7 elemenata.
6. Napisati JS program koji pronalazi par elemenata (indeksa dva broja) u datom nizu čiji je zbir jednak određenom ciljnom broju. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Zadajte proizvoljan ciljni broj.
7. Napisati JS program koji pronalazi presek dva niza. Kreirajte sami dva proizvoljna niza od minimum 7 elemenata.
8. Napisati JS program koji pronalazi par elemenata (indeksa dva broja) u datom nizu čiji je zbir jednak određenom ciljnom broju. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Zadajte proizvoljan ciljni broj.
9. Napisati JS program koji uklanja zadat element iz niza. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Zadajte element koji je potrebno ukloniti.
10. Napisati JS program koji uzima dva niza iste dužine i sabira vrednosti na istim pozicijama i upisuje ih u novi niz. Kreirajte sami dva proizvoljna niza od minimum 7 elemenata.
11. Napisati JS program koji pronalazi razliku dva niza (elementi prvog niza koji se ne pojavljuju u drugom nizu). Kreirajte sami dva proizvoljna niza od minimum 7 elemenata.
12. Napisati JS program koji pomera element niza sa jedne na drugu poziciju. Kreirajte sami proizvoljan niz od minimum 7 elemenata. Zadajte sa koje pozicije i na koju je potrebno pomeriti element.
13. Napisati JS program koji pronalazi jedinstvene elemente u oba niza. Kreirajte sami dva proizvoljna niza od minimum 7 elemenata.
14. Napisati JS program koji pronalazi sve permutacije elemenata niza (uključujući i duplikate). Kreirajte sami proizvoljan niz od 4 elemenata.
15. Napisati JS program koji pronalazi uniju dva niza. Kreirajte sami dva proizvoljna niza od minimum 7 elemenata.
