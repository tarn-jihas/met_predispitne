O predmetu CS105 - Osnove Web Tehnologija

CS105 - Osnove Web Tehnologija je ključni predmet za studente koji žele razumjeti i ovladati temeljima web razvoja. Ovaj kurs pruža sveobuhvatan uvod u osnovne tehnologije koje čine moderan web, uključujući HTML, CSS i JavaScript. Studenti će naučiti kako strukturirati web sadržaj, stilizirati web stranice i dodati interaktivnost koristeći klijentske skripte.

Tokom semestra, studenti će raditi na praktičnim projektima koji će im omogućiti da primijene naučene koncepte u stvaranju responzivnih i dinamičkih web stranica. Predmet također pokriva osnove web dizajna, pristupačnosti i optimizacije performansi. CS105 priprema studente za naprednije kurseve u web razvoju i daje im vještine potrebne za početak karijere u ovom brzo rastućem polju.
